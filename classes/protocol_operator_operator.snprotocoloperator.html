Class SNProtocolOperator

w An operator is responsible for performing crypto operations, such as generating keys and encrypting/decrypting payloads. Operators interact directly with platform dependent SNPureCrypto implementation to directly access cryptographic primitives. Each operator is versioned according to the protocol version. Functions that are common across all versions appear in this generic parent class.
